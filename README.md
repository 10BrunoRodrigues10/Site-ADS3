# Sobre o Projeto

Este projeto é um marketplace desenvolvido para simular uma loja de vendas. A aplicação permite exibir produtos, realizar buscas, filtrar e adicionar itens a um carrinho de compras.

A ideia foi simular o funcionamento de uma loja online, com foco na organização dos elementos na tela.

# Funcionalidades

Tem as seguintes funcionalidades:

- Listagem de produtos
- Busca de produtos por nome
- Filtrar por Categorias
- Adicionar produtos ao carrinho
- Cálculo do valor total
- Limpar carrinho

# Responsividade

O site foi desenvolvido utilizando CSS Grid e Media Queries, permitindo que o site se adapte a diferentes tamanhos de tela, como computadores, tablets e celulares.
